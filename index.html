XO

The Go-to library for creating games and desktop apps!

What is it ?

XO is a library aimed at Designers, Gamers, Developers and Go Beginners with the intention of easing the hassle of application development in Go.

XO provides packages for:

  • File input/output
  • 2D Graphics (3D Coming soon!)
  • Animation
  • Physics
  • GUI
  • HTML & CSS Styling
  • User Interaction
  • Windowing
  • OS Utilities
  • A/V
  • External devices
  • And more...

XO users aren't forced to choose between speed and simplicity because XO is based on high-performance libraries such as GLFW and OpenGL!

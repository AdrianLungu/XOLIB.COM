A library for creating desktop apps and games in Go.

XO is a library aimed at Designers, Gamers, Developers and Go Beginners with the intention of easing the hastle of application development in Go. XO provides packages for file input/output, 2D Graphics (3D coming soon!), Animation, Physics, GUI, HTML & CSS Styling, User Interaction, Windowing, OS Utilities, AV, External devices and more. XO Users aren't forced to choose between speed and simplicity because XO is based on high-performance libraries such as GLFW and OpenGL!
